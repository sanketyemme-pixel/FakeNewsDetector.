# AI-Powered Fake News Detection System.
This project uses machine learning techniques to classify news headlines as fake or real. The application was built to explore practical NLP and AI concepts through an interactive web interface.

# Features

- Fake vs Real news prediction
- Interactive user interface
- Prediction confidence score
- Beginner-friendly AI workflow
- Streamlit web application. 

# Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLP.

# Future Improvements

- Multi-language support
- Improved prediction accuracy
- Real-time news verification
- Advanced NLP integration
